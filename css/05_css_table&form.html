테이블 관련 css들

  • border : 테이블의 테두리와 셀의 테두리 지정(두가지 전부 지정해야한다)
  • border-collapse : 테이블의 테두리와 셀 테두리를 합칠지 여부를 설정
  • empty-cell : 테이블 테두리 설정을 분리시켜준 상태에서 내용이 없는 빈 셀들의 표시 여부를 설정
